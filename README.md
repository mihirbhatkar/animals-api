# animals-api

## API endpoints
<pre>
Get - /api/:id        =>   (get specific animal)
GET - /api            =>   (get all animals)
POST - /api           =>   (create an animal)
PUT - /api/:id        =>   (update an animal's data)
DELETE - /api/:id     =>   (delete a specific animal)
DELETE - /api/clear   =>   (delete all animals)
</pre>
